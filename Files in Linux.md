### Important files in Linux :



1. / -> The top-level directory. Every file and folder in Linux starts from here.

   ---
2. /usr -> Contains most user applications, commands, libraries, documentation, and shared files.

   ---
3. /boot -> Contains the files needed to start (boot) the Linux operating system.

   ---
4. /dev -> Represents hardware devices such as hard disks, USB drives, keyboards, mice, and terminals as files.

   ---
5. /etc -> Stores configuration (settings) files for the operating system and installed applications.

   ---
6. /home -> Contains the personal files and folders of normal users. 
Example: /home/your\_username.

   ---
7. /lib -> Contains library files required by programs and system commands to run.

   ---
8. /media -> Used to access removable devices like USB drives, CDs, and DVDs after they are mounted.

   ---
9. /opt -> Stores additional software installed manually or from third-party vendors.

   ---
10. /proc -> A virtual filesystem that provides information about running processes, CPU, memory, and the Linux kernel

    ---
11. /root -> The personal home directory of the root i.e administrator user.

    ---
12. /bin -> Contains basic commands like ls, cp, mv, cat, and pwd that all users can use.

    ---
13. /sbin -> Contains system administration commands such as shutdown, reboot, fdisk, and mkfs. These are mainly used by the root user.

    ---
14. /tmp -> Stores temporary files created by programs. These files are usually deleted automatically after a reboot.

    ---
15. ##### /var -> Stores data that changes frequently, such as log files, mail, cache, print queues, and databases.





### In Simple Words :



1. ##### bin → Binary commands
2. ##### boot → Boot files
3. ##### dev → Devices
4. ##### etc → Configuration (settings)
5. ##### home → User files
6. ##### lib → Libraries
7. ##### media → USB mount point
8. ##### opt → Optional software
9. ##### proc → Process information
10. ##### root → Root user's home
11. ##### sbin → System binaries
12. ##### tmp → Temporary files
13. ##### usr → User programs
14. ##### var → Variable data (logs, cache, mail)
