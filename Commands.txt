Important Commands for Devops :-

1) ls -> List the files and directories
2) ls -l -> Detailed listing of files and directories i.e instead listing only the file names it lists with permissions , date , time etc
3) ls -a -> It is used to list the files and folders including Hidden files
4) ls -al -> Detailed listing of files and directories with Hideen files
5) man <command> -> It is used to display the details of commands
6) mkdir <directory name> -> It is used to create new directories
7) rmdir <directory name> -> It is used to remove directories
8) cp <file1 name> <file2 name> -> It is used to copy/move files
9) pwd -> It displays the present working directory
10) cd -> It is used to change directory
11) cd .. -> It is used to come back to the previous directory
12) cd ../.. -> It is used to come back to the two directories back from the present working directory
13) clear -> Clear the screen
14) rm -> It is used to remove the files/folders if it is empty
15) rm -rf -> It is used to remove the files/folders even it contains some files inside it
16) touch -> It is used to create a file
17) vi and nano -> These both are text editors , where we can edit the files
18) && -> It is used to perform multiple commands at a time
    For example : touch file1.txt && vi file1.txt
19) sudo apt -> It is used to install , update , delete packages , Here apt is package manager for Ubuntu
20) ping <host name> -> It is used to check the network connectivity , server can be reached or not etc
21) nslookup <host name> -> It is used to display the details of remote host 
22) wget <url> -> It is used to install files provided in the online , The files can be installed using url
23) ip addr -> It displays the ip address and network interfaces of your own computer
24) curl <url> -> It is used to display the html code of a website
